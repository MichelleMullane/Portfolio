Draw Your Face Off

Draw Your Face Off! is a collaborative drawing app where guests and logged-in users can draw on a canvas with an array of tools and colors. Users generate links to invite others to their drawing rooms where they can collaborate on one canvas. A leader feature allows logged-in users to create groups and post messages to the group drawing rooms with on-demand monitoring of each group's work.

Green Garden

Green Garden is a mock e-commerce site where guests and logged-in users can browse beautiful plants and trees, add to a shopping cart, and complete a purchase. The shopping cart page allows users to update the items in the cart, change the quantities of items in the cart, delete items from the cart, and view a cart subtotal. Administrative users have unique access where they can view registered users and update the store inventory.

Com Pear

Com Pear helps users to make informed purchases. Users can create comparison projects (called pears) and compare features of any desired item. Data visualizations via Chart.js allow the user to make efficient comparisons of numerical data.
